# homebrew-muckdb

Homebrew tap for [muckdb](https://github.com/nickkaltner/muckdb) — a facade over
the duckdb CLI with a live web view.

```sh
brew install nickkaltner/muckdb/muckdb
```

`Formula/muckdb.rb` is generated and committed automatically by muckdb's release
workflow on each tagged release. Don't edit it by hand.
